# Basic Calculator With Windows Forms

A simple desktop calculator application built with C# and the Windows Forms framework in Visual Studio. This project was created to practice C# fundamentals and event-driven programming.

The application takes two numbers from the user and performs a basic arithmetic operation based on the button clicked.
